CCL is a minimal language for defining shared data models and generating code from them.

This docs section is intentionally short on ceremony:

- Read the getting-started page if you want the fastest path from install to first generated output.
- Read the language overview if you want to understand the shape of the language before using it.
- Jump to reference pages when you already know the workflow and just need specifics.
