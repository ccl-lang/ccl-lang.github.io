This page is the quickest way to answer a simple question: _can CCL target my language yet?_

Statuses are intentionally blunt:

- **Supported** means the generator is usable now.
- **Developing** means the generator exists as active work, but the feature set is not finished.
- **Planning** means the target is on the roadmap, but no current generator support should be assumed.

## Notes

- Go and GDScript are the currently supported targets.
- C#, Python, JavaScript, and TypeScript are active development targets.
- C, C++, Rust, and Lua are planned targets.
