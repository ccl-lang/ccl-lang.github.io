---
title: JSON Attributes
description: Attributes that control generated JSON field names.
---

## `JsonPropertyName`

Overrides the generated JSON property name for a field.

```txt
model UserInfo {
    [JsonPropertyName("user_id")]
    Id: string;
}
```

This attribute is field-local.

## `JsonPropertyNamingStrategy`

Applies a naming strategy to generated JSON field names for a model or scope.

```txt
#[JsonPropertyNamingStrategy("snake_case")]
```

Supported strategy names are:

- `PascalCase`
- `camelCase`
- `snake_case`
- `kebab-case`

`JsonPropertyName` on a field takes precedence over the naming strategy.
