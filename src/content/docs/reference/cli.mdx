## Generate

```bash
ccl generate --source definitions.ccl --output path/to/output --language Go
```

## Main inputs

- `--source`: path to the `.ccl` file.
- `--output`: directory where generated files should be written.
- `--language`: target language for generation.

## Notes

- Keep generated output in a dedicated directory.
- Treat the `.ccl` file as the source of truth and generated files as artifacts.
